Bonjour,

voici un ensemble de scripts python écrits dans le cadre du cours d'interfaces haptiques. Les scripts ont été testés avec un Novint Falcon

vous trouverez dans le dossier /src une liste de scripts exécutant chacun des exemples traités dans le rapport :

- ressort.py fut utilisé pour caractériser la masse de l'effecteur

- Mur_y_simple.py fait office de rendu pour la partie du mur simple, dans la définition de la démarche de retour haptique adéquate

- plan_incline_epais.py traite le cas du plan incliné avec épaisseur ajoutée

- sphere.py traite du retour haptique pour la sphère

- glissiere.py execute une simulation d'une glissière rectiligne sans cran

- viscosite.py execute une simulation de l'effecteursoumis à des frottements visqueux


(les scripts utils et vect3d sont necessaires au bon fonctionnement des scripts mentionnés ci-dessus)


un folder Blender est présent avec les librairies necessaires au bon fonctionnement du projet blend.
